¿Cómo Registrar un Reclamo?

1) Dirígase a la sección Reclamo > Registrar e ingrese los datos solicitados en el orden en que aparecen (area de servicio, reclamo, barrio, etc).


2) Una vez ingresados los datos obligatorios (*), presionando el botón Siguiente, el sistema valida que el Reclamo que intenta registrar no halla sido ingresado con anterioridad. En caso de que ya se encuentre registrado, le aparecerá el siguiente mensaje:


Caso contrario, si el Reclamo no ha sido ingresado con anterioridad, se le mostrará un resumen con los datos del mismo:


3) Antes de enviar el Reclamo, corrobore que los datos ingresados sean correctos, ya que una vez registrado el mismo, sus datos no pueden modificarse. Tilde el casillero Confirmo haber revisado los datos del reclamo y presione Enviar.


El sistema automáticamente le envía un correo electrónico con el código del Reclamo para su posterior consulta. Le recomendamos no borre este email para poder seguirlo paso a paso.
