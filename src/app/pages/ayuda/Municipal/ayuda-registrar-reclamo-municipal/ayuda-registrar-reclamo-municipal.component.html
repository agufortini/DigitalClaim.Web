¿Cómo Registrar un Reclamo?

Dirígase a la sección Reclamo > Registrar

1) El sistema solicitará el ingreso de los datos del Ciudadano Reclamante:


Una vez ingresado los mismos, mediante el botón Siguiente, el sistema valida que los datos del usuario no existan, y si existen, muestra el siguiente mensaje y cancelará el proceso:


En caso que el usuario sea nuevo, dará paso al ingreso de los datos del Reclamo:


Si desea modificar algún dato del usuario ciudadano, puede regresar al formulario anterior con el botón Anterior.

2) Una vez ingresados los datos obligatorios (*), presionando el botón Siguiente, el sistema valida que el Reclamo no halla sido ingresado con anterioridad. En caso de que ya se encuentre registrado, le aparecerá el siguiente mensaje (en caso de estar en estado "Sin Asignar"):


Caso contrario, si es un Reclamo nuevo, se le mostrará un resumen con los datos del mismo:


Si desea modificar algún dato del usuario ciudadano, puede regresar al formulario anterior con el botón Cancelar.

3) Antes de enviar el Reclamo, corrobore que los datos ingresados sean correctos, ya que una vez registrado el mismo, sus datos no pueden modificarse. Tilde el casillero Confirmo haber revisado los datos del reclamo y presione Enviar.


El sistema automáticamente creará un usuario al ciudadano reclamante, el cual podrá utilizar posteriormente ingresando como nombre de usuario y contraseña su DNI. Luego, se registra el Reclamo y se le envía un correo electrónico con el código del mismo para que pueda consultar las actividades llevadas a cabo.
